# **国富论**
## *前言*
&emsp;&emsp;亚当·斯密是18世纪英国著名的经济学家和伦理学家。1768年开始撰写《国富论》，1773年完成初稿，之后他又花费三年时间润色此书。1776年3月，《国富论》出版，其影响之广泛，遍布英国本土以及整个欧洲美洲。

&emsp;&emsp;《国富论》全名为《国民财富的性质和原因的研究》。书中总结了近代初期各国资本主义发展的经验，批判吸收了当时的重要经济理论，对整个国民经济的运转过程做了系统的描述，为经济学确定了完整的框架。它标志着经济学作为一门独立学科的诞生，是现代政治经济学研究的起点。因此，《国富论》被誉为“第一部系统的伟大的经济学著作”、“经济学百科全书”、“西方经济学圣经”，而亚当·斯密则赢得了“现代经济学之父”和“自由企业的守护神”的美誉。

&emsp;&emsp;《国富论》奠定了资本主义自由经济的理论基础，第一次提出了市场经济会由“看不见的手”自行调节的理论。后来的经济学家大卫·李嘉图进一步发展了自由经济自由竞争的理论；马克思则从中看出自由经济产生周期性经济危机的必然性，提出用计划经济理论解决的思路；凯恩斯则提出政府干预市场经济宏观调控的方法。

&emsp;&emsp;虽然《国富论》中的许多学说被后来的经济学家所超越，但是亚当·斯密所确立的研究方向和方法影响深远，他富有预见性的洞察和许多明确、实用的见解，总能引起后人的重新思考。他在封建欧洲解体之末，近代世界开始之时著作。在当时的世界中，封建制度仍以既得利益集团经常表现的顽固性在坚持。他正是为反对这种利益集团而动笔的。结果是《国富论》对经济理论和国家政策产生了深刻的影响，从而形成了我们今天身处其中的整个生活环境。相信不同人阅读《国富论》会有不同的体会、不同的收获。
## *国民财富本质和原因的研究*
* ### **第一篇** 论劳动生产力逐步提高的原因，以及劳动产品在不同阶层之间自然分配的顺序
    - ### **第一章** 论劳动分工
        1. ### 劳动分工给劳动生产力带来了极大的提升，并且影响了绝大部分用于指导和实际使用的技能、技巧以及判断力。
            - #### 举个例子——别针的制造
                #### `如果一个没有接受过培训的工人来制造别针，相信一天之内最多能生产20枚别针(相信已经拼尽全力)。但是把制造过程拆分成18道工序。第一个人抽丝，第二个人拉直，第三个人切断，第四个人削尖，第五个人磨平顶端以便安装针头，第六个人...；那么做针头就有两三步操作；安装针头是一项专门的业务，刷白针头、甚至是把针头装进纸盒中又是一项专门的业务。这18道工序分别由10到18人的团队协作完成，人少的时候一人可能要身兼多职。在这样的分工情况下，他们一天最多差不多能制造4.8万枚别针，也就是人均最多可以生产4800枚，远远超出单人的20枚。提升了上百倍的生产力。`
        2. ### 无论哪种行业，只要劳动采用了分工，生产力就能取得相应的增长。各种行业和职业的分立也是这一好处的结果。产业与生产力大幅增加的国家，其各种行业的分立一般也极其细化。
        3. ### 劳动分工使得同样数量的人所能完成的工作量大大增加，这要归因于三种情况：
            - #### 每一个劳动者的熟练程度提高。
                #### `劳动者熟练程度的提高势必增加他能完成的工作量。分工的结果使每个人的工作简化成某种简单的操作，并使这种操作成为劳动者终生从事的唯一职业，这也必然大大提高他的熟练度。`
            - #### 节约了由一种工作转换到另一种工作通常损耗的时间。
                #### `这种时间的节省有可能比我们想象的要大得多。劳动者不可能从一种工作很快的转到另一种在不同地点、使用不同工具的工作。比如一个农村织布匠从织布机转换到田野，又从田野转换到织布机，这必然要浪费大量的时间。即便是两种工作在同一地点，效果也不见得会好到哪儿去。如果一个人每天要在20个工种之间来回切换，那么他必然会养成闲荡和漫不经心的习惯，几乎总是懒懒散散，就算最紧迫的情况下也不会让他全神贯注。`
            - #### 发明很多机器，用于便利和简化劳动，使一个人能干更多人的活。
                #### `使用"适当"的机器能在很大程度上简化和方便劳动。当一个人注意力全部集中在单一目标，而不是分散在许多事物上时，他们就有可能更容易、更迅捷地达到目的。我们甚至可以预期，从事每一种具体劳动的劳动者中总会有些人过不了多久就能找出比较容易和迅速的方法去完成工作，只要工作的性质允许他们做出改进。最初使用蒸汽机时，工厂常常雇佣一个男孩，他在活塞上下运动时负责开关汽锅与汽缸之间的通道。一个贪玩的孩子注意到如果用绳子把开关通道阀门的柄系在机器的另一部分上，阀门就能自动开关无需值守。这样他就可以和其他伙伴自由玩耍。`
        4. ### 机器的改良很多时候不是出自有机会使用机器的人。
            - #### 当制造机器成为行业，那么机器制造者会用尽聪明才智去改良它。
            - #### 还有一些改良则来自于哲学家或者思想家。他们不是通过具体做某一件事情，而是观察每一件事情——把两件看起来毫不相干的事儿联系起来。
            - #### 每个人都变得对他自己那个特殊部分的工作更加在行，就整体而言也就完成了更多的工作，那么用科学手法作统计的工作量也由此大大增加。
        5. ### 分工使得各种行业的产量倍增，普遍的富裕可以惠及最底层的劳苦大众。
            - #### 除了满足自身需求，一种产品的制造者还有大量产品可以销售；另外一种产品的制造者的情况也基本相同。如果需要，每个劳动者都能以自身生产的大量产物换得其他劳动者生产的大量产物，或换得与产品等值的货币。他能充分供给别人所需的物品；他自身所需的也能得到别人的充分供给。
        6. ### 在一个文明发达的国家中，如果看一下最普通的工匠使用的生活用品，你会知道为了使他们能享用它，必须贡献自己工作的一小部分（哪怕只是很小的一部分）的人多得数不胜数。
            - #### 假如我们考察一下所有我们使用的物品，看看每种物品要使用多少不同的劳动，我们就会明白，没有成千上万人的协同合作，一个文明社会中最普通的工人也不可能得到他最简易的生活用品。
